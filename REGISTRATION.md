# PayCrawl Hackathon Registration

Hackathon: `agentic-payments-defai`

Register immediately after the public GitHub repository exists. Registration creates a private draft and returns the assigned `celo_...` attribution tag; it does not publish the final submission.

## Registration-stage inputs

- Builder name
- Builder email used for Google sign-in
- Social handle
- Team name
- Agent name
- Personal Telegram handle
- Public GitHub repository URL
- Dedicated Celo mainnet agent/payTo wallet address

Never provide a private key, seed phrase, repository credential, or other secret.

## Tracks

```json
[

```

## Applicable bounties

```json
[
  "most-revenue-generated-1st",
  "most-revenue-generated-2nd",
  "most-x402-payments-1st",
  "most-x402-payments-2nd",
  "askbots-prize-pool",
  "track-4-prize"
]
```

First and second place in the same track are mutually exclusive. Cross-track prize stacking is not explicitly confirmed in the published rules and should be confirmed with the organizers.

## Final-submission inputs

- One-line tagline and complete project description
- Live demo URL
- Video URL, if produced
- Public X/Twitter submission post URL
- Celo mainnet ERC-8004 identity URL
- Celo network: `celo-mainnet`
- Contract addresses, only if PayCrawl deploys contracts
- Agent contribution notes
- Aigora profile URL and Aigora feedback issue URL for the Aigora track

The project remains a private draft until the builder explicitly approves publication.
